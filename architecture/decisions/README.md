## Template

```markdown
# ADR-NNN: Title

## Status
[Proposed | Accepted | Deprecated | Superseded]

## Context
What problem are we solving?

## Decision
What did we decide?

## Consequences
### Positive
- Benefits

### Negative
- Drawbacks
```

---

## Creating New ADR

1. Copy template
2. Name: `ADR-NNN-short-title.md`
3. Fill all sections
4. Add to index
5. Submit for review
